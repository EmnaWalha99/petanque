 # Petanque Application

 ## Overview
 Simple Next.js app to manage pétanque matches (add, view, delete).

 ## Setup
 ```bash
 npm install
 npm run dev

 Open http://localhost:3000
 ```

 ## Structure
 pages/ → app pages (login, matches)
 components/ → MatchForm, MatchList
 parseConfig/ → Parse setup

 ## Features
 - Login / Signup
 - Add match
 - View matches
 - Delete match

 ## Scripts
 - npm run dev
 - npm run build
 - npm start

 ## Notes
 Uses Parse as backend
